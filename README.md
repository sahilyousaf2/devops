# Git Basics

## What is Git?
Git is a distributed version control system used to track changes in code.

## Key Commands

### Initialize a Repository
```bash
git init
```

### Clone a Repository
```bash
git clone <repository-url>
```

### Check Status
```bash
git status
```

### Stage Changes
```bash
git add <file>
git add .
```

### Commit Changes
```bash
git commit -m "Your message"
```

### Push to Remote
```bash
git push origin main
```

### Pull from Remote
```bash
git pull origin main
```

## Basic Workflow
1. Make changes to your files
2. Stage changes with `git add`
3. Commit with `git commit`
4. Push with `git push`
